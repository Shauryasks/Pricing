# Pricing Table (Using HTML and CSS)

##About
A clean and simple 3-plan pricing table made using HTML and CSS Flexbox. It shows Basic, Standard and Premium plans with storage, user limits, and support options.

##Usages
1) Open 02_Pricing.html in a browser to view.
2) Edit 02_Pricing.html to change text and prices.
3) Edit 02_Pricing.css to adjust colors, spacing and layout.
